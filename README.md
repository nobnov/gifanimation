# Make gif animation from multiple images

## Usage

	$ pip install -r requirements.txt
(Clone from repository)
	
**or**

	$ pip install Pillow
(Manually install)
	
## Sample usage

	$ python convert.py

## Result

generate jpg_to_gif.gif
